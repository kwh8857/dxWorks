DX Digital ID
디엑스 디지털 ID
디엑스 디지털 아이디는 블록체인 기반의
분산신원증명(DID) 기술로 구현된
차세대 모바일 신분증 서비스입니다.
사원증, 학생증, 방문증 등 다양한 신분증을 안전하게 통합하고,
QRCode, NFC, GPS, 안면인식을 통한 인증 기능으로
빠르고 정확한 신원 확인이 가능합니다.

ID를 공개하고 표시할 수 있습니다.
